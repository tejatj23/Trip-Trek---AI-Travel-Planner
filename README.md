# Trip-Trek---AI-Travel-Planner
TripTrek is an intelligent travel planning application that generates personalized travel itineraries using Artificial Intelligence. The system analyzes user inputs such as destination, travel duration, budget, and interests to automatically create a customized day-wise travel plan. It integrates Generative AI models to provide smart recommendations for destinations, transportation, accommodations, and activities. 

The application is built using Python, Streamlit, and Google Generative AI (Gemini/PaLM) to deliver an interactive user experience. Users can also interact with an AI chatbot to ask travel-related questions, get real-time suggestions, and refine their itinerary.
TripTrek simplifies the traditionally complex trip planning process by combining itinerary generation, cost optimization, and travel recommendations into a single platform, making travel planning faster, smarter, and more personalized.
